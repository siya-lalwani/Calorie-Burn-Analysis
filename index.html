Calories Burn Analysis

An end-to-end data science project — EDA, feature engineering & regression modelling

Project Overview

  • Explores fitness data to analyze how workout types, gender, body metrics, and heart rate influence calories burned.
  • Uses visualization and machine learning to reveal key patterns affecting calorie expenditure.
  • Includes both regression (predicting calories) and classification (categorizing burn levels).

Repository: github.com/siya-lalwani/calorie-burn-analysis

Objectives

  • Understand relationships between workout parameters and calorie burn.
  • Visualize calorie distribution, workout comparison, and gender-based trends.
  • Generate correlation heatmaps to identify numeric dependencies.
  • Build:
    • Random Forest Regressor for calorie prediction.
    • Random Forest Classifier for calorie burn categorization.
  • Identify key predictors using feature importance analysis.

Technologies Used

  • Python
  • Pandas, NumPy — Data cleaning & manipulation
  • Matplotlib, Seaborn — Visualization
  • Scikit-learn — Modeling & evaluation

Key Visualizations

Distribution of Calories Burned

Calories Burned by Workout Type

Interpretation
  • Cardio and HIIT show highest medians, confirming efficiency for calorie burn.
  • Yoga and Stretching display smaller calorie ranges — low-intensity workouts.

Calories Burned vs Average BPM

Interpretation
  • Positive correlation: higher heart rates lead to higher calories burned.
  • Longer sessions (larger points) confirm both duration & intensity matter.
  • HIIT & Cardio dominate high-BPM zones — validating physiological patterns.

Feature Importance Plot

Interpretation

  • Confirms both physiological and workout-related features drive predictions.

Model Metrics & Interpretation

Algorithm: Random Forest Regressor
Workflow: 80/20 train-test split (random_state=42), trained and evaluated using R² and MAE.

R² Score
Measures how much variance is explained (1.0 = perfect)
MAE
Average absolute difference between predicted & actual calories
Interpretation
  • High R² (≈0.85–0.9) → model explains most calorie-burn variance.
  • Low MAE (≈20–30) → predictions are close to real-world values.

Algorithm: Random Forest Classifier
